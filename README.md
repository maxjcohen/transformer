## Installation

```terminal
.\scripts\init-env.ps1
```

## Usage

```python
from flights_time_series_dataset import FlightsDataset
from time_series_predictor import TimeSeriesPredictor
from tst import Transformer

tsp = TimeSeriesPredictor(
    Transformer(),
    max_epochs=50,
    train_split=None,
)

tsp.fit(FlightsDataset())
```
